# Drawdio

Drag-and-drop mockup tool for audio plugin UIs. Design knobs, sliders, meters, sequencers, and more — then export as PNG, SVG, or JSON for implementation in JUCE, nih-plug, iPlug2, or any framework.

## Quick start

Open `drawdio.html` in any modern browser. No install, no dependencies, no build step.

## Features

- **Component palette** — rotary knobs, sliders, buttons, meters, step sequencer, spectrum analyzer, and more
- **Snap-to-grid** — configurable grid with optional free placement (hold Alt)
- **Properties panel** — edit position, size, color, and type-specific parameters
- **Canvas presets** — Full Synth (900x600), Compact Effect (400x300), and custom sizes
- **Export** — PNG (1x/2x/3x), SVG, JSON layout with full component data
- **Effects** — drop shadow, gloss, texture fills, gradients, and more
- **Keyboard shortcuts** — Ctrl+Z undo, Ctrl+D duplicate, G toggle grid, and more
- **Save/Load** — `.drawdio.json` files preserve your full layout

## Usage with AI

Export your mockup as PNG (screenshot) or JSON (structured layout), then share it with Claude, ChatGPT, or any AI assistant to guide plugin UI implementation. The JSON format includes exact positions, sizes, types, and properties for every component.
